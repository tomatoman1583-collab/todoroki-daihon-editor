# 轟 台本エディタ（公開版）

Mac・Windows どっちのパソコンでも、ブラウザで開いて使える台本エディタ。
開いたパソコンの中のファイルを読み込んで、読み上げ・26字超過チェック・句点チェックができる。

## 公開URL（これを家族で共有する）

**https://tomatoman1583-collab.github.io/todoroki-daihon-editor/**

- 奥さんのMacBookでも、このURLを開くだけで使える。
- 同じURLで、それぞれのパソコンの中のファイルを読み込める（自分のPCのファイルしか読まないので安全）。

## 使うときのコツ

- **おすすめブラウザは Chrome**。
  - Chrome / Edge：「台本を開く」で選んだファイルに、そのまま「保存」で上書きできる。
  - Safari：読み込み・読み上げ・チェックは全部動くが、「保存」は**ダウンロード**になる（元ファイルに置き換えてね、という形）。

## 中身について

- 公開しているのは `index.html` 1枚だけ（このエディタの画面）。
- 台本などの中身は公開されない。各自のパソコンの中のファイルを、その場で読み込むだけ。

## 更新のしかた（マサト用・Windows）

エディタ本体（`../台本エディタ.html`）を直したら、この公開版にも反映する：

- `更新する.ps1` を実行する（または Claude に「公開版を更新して」と言う）。
- 1分ほどで上のURLに反映される。

※ `更新する.ps1` と GitHub CLI のパスは **Windows専用**。Macから更新する場合は手順の調整が必要。
